# Metabolic Fitness - Plataforma de Fisiología Clínica del Ejercicio

Plataforma líder mundial en Fisiología Clínica del Ejercicio. Combina tratamientos clínicos basados en evidencia con capacitación profesional de excelencia.

## Stack Tecnológico

- **Framework**: Next.js 14 (App Router)
- **Estilos**: Tailwind CSS
- **Animaciones**: Framer Motion
- **Iconos**: Lucide React
- **Fuentes**: Google Fonts (Montserrat y Open Sans)

## Guía de Estilo

### Colores
- **Fondo**: Blanco Puro (#FFFFFF)
- **Texto Principal**: Gris Carbón (#404041)
- **Primary**: Verde Lima (#5CB85C)
- **Secondary**: Azul Cian (#00AEEF)
- **Fondo Secundario**: Gris Muy Pálido (#FAFAFA)

### Tipografía
- **Títulos**: Montserrat (Bold/Extra Bold)
- **Cuerpo**: Open Sans (Regular/Light)

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Build

```bash
npm run build
npm start
```
